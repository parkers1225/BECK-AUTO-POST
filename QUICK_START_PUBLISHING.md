# Quick Start: Publish to Chrome Web Store

## 🚀 Fast Track (30 Minutes)

### Step 1: Test Package (5 min)

1. Open Chrome: `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select: `beck-auto-post-extension-package` folder
5. Verify extension loads and works

✅ **If it works, proceed to Step 2**

---

### Step 2: Host Privacy Policy (10 min)

**Option A: GitHub Pages (Free, Easy)**
1. Create GitHub repository
2. Upload `privacy-policy.html`
3. Enable GitHub Pages
4. Get URL: `https://yourusername.github.io/repo/privacy-policy.html`

**Option B: Your Website**
1. Upload `privacy-policy.html` to your website
2. Get URL: `https://yourwebsite.com/privacy-policy.html`

**Option C: Use Template**
- Privacy policy HTML file created: `privacy-policy.html`
- Customize with your contact info
- Host anywhere (even free hosting works)

✅ **Once you have the URL, proceed to Step 3**

---

### Step 3: Take Screenshot (5 min)

1. Open extension
2. Load some vehicle data
3. Take screenshot (Windows: Win+Shift+S)
4. Crop to 1280x800
5. Save as PNG

**Minimum:** 1 screenshot required  
**Recommended:** 3-5 screenshots

✅ **Once screenshot ready, proceed to Step 4**

---

### Step 4: Create Chrome Web Store Account (10 min)

1. Go to: https://chrome.google.com/webstore/devconsole
2. Sign in with Google account
3. Click "Pay Registration Fee" ($5 one-time)
4. Complete profile

✅ **Account created, proceed to Step 5**

---

### Step 5: Upload & Submit (15 min)

1. Click "New Item"
2. Upload: `beck-auto-post-extension-v1.0.0.zip`
3. Fill in store listing:
   - Name: `Beck Auto-Post`
   - Description: (from `STORE_LISTING_DESCRIPTION.md`)
   - Privacy Policy URL: (from Step 2)
   - Screenshot: (from Step 3)
   - Icon: `icons/beck-logo.png`
   - Visibility: **Unlisted**
4. Click "Submit for Review"

✅ **Submitted! Wait 1-3 days for approval**
