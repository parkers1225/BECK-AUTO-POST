🚗 Create Extension Icons

This tool will generate simple placeholder icons for your Chrome extension.

Instructions:
  1. Click "Generate Icons" to create the icon images
  2. Right-click each icon preview and select "Save image as..."
  3. Save them in the icons folder with these exact names:
    • icon-16.png
    • icon-48.png
    • icon-128.png
