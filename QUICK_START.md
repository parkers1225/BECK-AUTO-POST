# 🚀 QUICK START GUIDE

Follow these 5 simple steps to get your extension running:

## ✅ Step 1: Create Icons (2 minutes)

**Easiest Method:**
1. Double-click `create-icons.html` to open it in your browser
2. Click "Download All Icons" button
3. Move the 3 downloaded PNG files to the `icons` folder:
   - `icon-16.png` → `icons/icon-16.png`
   - `icon-48.png` → `icons/icon-48.png`
   - `icon-128.png` → `icons/icon-128.png`

**Done!** ✅

---

## ✅ Step 2: Load Extension in Chrome (1 minute)

1. Open Chrome
2. Type `chrome://extensions/` in the address bar
3. Turn ON "Developer mode" (top-right toggle)
4. Click "Load unpacked"
5. Select the `FB MARKETPLACE PROJECT` folder
6. Extension should appear! ✅

---

## ✅ Step 3: Get API Key (5 minutes)

**Choose one:**

### Option A: OpenAI (Easier)
- Go to: https://platform.openai.com/api-keys
- Sign up/login
- Click "Create new secret key"
- Copy the key

### Option B: Anthropic Claude
- Go to: https://console.anthropic.com/
- Sign up/login
- Create API key
- Copy the key

---

## ✅ Step 4: Configure Extension (1 minute)

1. Click the extension icon in Chrome toolbar (🧩 if not visible)
2. Click the ⚙️ settings button
3. Select your AI service
4. Paste your API key
5. Click "Save Settings" ✅

---

## ✅ Step 5: Test It! (5 minutes)

1. **Upload CSV:**
   - Click extension icon
   - Click "Choose CSV File"
   - Select your vehicle CSV

2. **Select Vehicle:**
   - Click "Load Photos from CSV" or "Upload Local Photos"
   - Click on a vehicle photo

3. **Generate Description:**
   - Enter a prompt like: "Create an engaging description"
   - Click "Generate Description"
   - Wait for result ✅

4. **Post to Marketplace:**
   - Click "Open Facebook Marketplace"
   - Click "Fill Marketplace Form"
   - Review and submit! ✅

---

## 🎉 You're Done!

Your extension is ready to automate Facebook Marketplace listings!

## ❓ Having Issues?

See `NEXT_STEPS.md` for detailed troubleshooting.

**Most Common Issues:**
- ❌ Icons missing → Use `create-icons.html`
- ❌ Extension won't load → Check Developer mode is ON
- ❌ CSV not loading → Check VIN column exists
- ❌ AI not working → Verify API key is correct

---

**Need help?** Check the browser console (F12) for error messages.
