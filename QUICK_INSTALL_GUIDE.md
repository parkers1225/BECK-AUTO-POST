# Quick Install Guide - Beck Auto-Post Extension

## 🚀 5-Minute Installation

### Step 1: Extract the Extension
1. Download `beck-auto-post-extension-v1.0.0.zip`
2. Right-click → **Extract All...**
3. Choose location (e.g., `C:\Users\YourName\beck-extension`)
4. **Keep this folder - don't delete it!**

### Step 2: Install in Chrome
1. Open Chrome
2. Type: `chrome://extensions/` (press Enter)
3. Toggle **"Developer mode"** ON (top-right)
4. Click **"Load unpacked"** (top-left)
5. Select your extracted folder
6. Click **"Select Folder"**

### Step 3: Pin the Extension
1. Click puzzle piece icon (🧩) in Chrome toolbar
2. Find "Beck Auto-Post"
3. Click pin icon to keep it visible

### Step 4: Configure Your Store

Click extension icon → Settings (⚙️) → Enter your store info:

**BECK CDJR:**
- CSV URL: `http://10.26.1.230:3000/csv/beck-cdjr`
- API Key: `beck-cdjr-secret-key`

**BECK CHEVY:**
- CSV URL: `http://10.26.1.230:3000/csv/beck-chevy`
- API Key: `beck-chevy-secret-key`

**BECK FORD:**
- CSV URL: `http://10.26.1.230:3000/csv/beck-ford`
- API Key: `beck-ford-secret-key`

**BECK NISSAN:**
- CSV URL: `http://10.26.1.230:3000/csv/beck-nissan`
- API Key: `beck-nissan-secret-key`

✅ Enable auto-refresh  
✅ Set interval: 15 minutes  
✅ Click **"Save Settings"**

### ✅ Done!
You should see vehicles in the photo gallery!

## 🔧 Troubleshooting

**Extension won't load?**
- Make sure Developer mode is ON
- Select the folder with `manifest.json` inside

**Can't see extension icon?**
- Click puzzle piece (🧩) → Find "Beck Auto-Post" → Pin it

**"Failed to fetch CSV"?**
- Check you're on company network
- Verify URL and API key are correct
- Click "Allow" when permission prompt appears

**Need help?** Contact IT Support
