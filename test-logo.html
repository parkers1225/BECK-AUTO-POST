Logo Transparency Test

This page helps you verify if your logo has true transparency.

Test 1: Over Blue Gradient (Extension Header Style)

Test 2: Over Checkered Background (True Transparency Test)

Test 3: Over White Background

Instructions:

  • Test 1: Should show logo clearly over blue gradient (like in extension)
  • Test 2: If you see checkered pattern through the logo, it has transparency ✓
  • Test 3: If logo disappears or blends into white, it has transparency ✓
  • If you see a white/colored box around the logo in any test, the PNG doesn't have true transparency
