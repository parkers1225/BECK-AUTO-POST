# Screenshot Guide for Chrome Web Store

## Required Screenshots

Chrome Web Store requires at least **1 screenshot**, but you can upload up to **5 screenshots**.

### Screenshot Specifications

- **Format:** PNG or JPEG
- **Size:** 
  - Minimum: 1280x800 pixels
  - Alternative: 640x400 pixels
- **File Size:** Maximum 1MB per image
- **Content:** Must show the extension in action

## How to Take Screenshots

### Method 1: Windows Snipping Tool
1. Press `Windows Key + Shift + S`
2. Select area to capture
3. Save as PNG
4. Resize to 1280x800 if needed

### Method 2: Chrome Screenshot Extension
1. Install a screenshot extension
2. Capture full page or selected area
3. Save and edit

### Method 3: Browser Developer Tools
1. Press F12
2. Use device toolbar (Ctrl+Shift+M)
3. Set to 1280x800
4. Take screenshot

### Method 4: Third-Party Tools
- Lightshot
- Greenshot
- ShareX

## Editing Screenshots

### Recommended Tools:
- **Free:** GIMP, Paint.NET, Canva
- **Paid:** Photoshop, Snagit

### What to Edit:
- ✅ Crop to correct size (1280x800)
- ✅ Add annotations/highlights (optional)
- ✅ Blur sensitive information (API keys, etc.)
- ✅ Add text labels (optional)
- ✅ Optimize file size (< 1MB)

### What NOT to Do:
- ❌ Don't use fake/mock data that misrepresents functionality
- ❌ Don't show sensitive information (API keys, passwords)
- ❌ Don't use copyrighted images without permission

## Tips for Great Screenshots

1. **Show Real Usage**
   - Use actual vehicle data (anonymize if needed)
   - Show real interface, not mockups

2. **Highlight Key Features**
   - Use arrows or highlights to point out features
   - Add text labels if helpful

3. **Keep It Clean**
   - Remove unnecessary browser UI
   - Focus on extension interface

4. **Consistent Style**
   - Use similar styling across all screenshots
   - Maintain consistent branding

5. **Test Different Sizes**
   - Ensure screenshots look good at different sizes
   - Chrome Web Store may resize them

---

## Quick Screenshot Workflow

1. **Prepare Extension**
   - Load with sample data
   - Configure settings
   - Test all features

2. **Capture Screenshots**
   - Take 3-5 screenshots showing different features
   - Use 1280x800 resolution

3. **Edit Screenshots**
   - Crop to size
   - Add annotations (optional)
   - Blur sensitive info
   - Optimize file size

4. **Review**
   - Check all requirements met
   - Verify no sensitive data
   - Ensure clarity

5. **Upload**
   - Upload to Chrome Web Store
   - Verify they display correctly
