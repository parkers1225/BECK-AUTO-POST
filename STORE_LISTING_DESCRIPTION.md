## Detailed Description (Up to 16,000 characters)

### Full Description:

```
Beck Auto-Post - Automated Facebook Marketplace Vehicle Listing Tool

Streamline your dealership's Facebook Marketplace operations with Beck Auto-Post, an intelligent Chrome extension designed to automate vehicle listing creation.

KEY FEATURES:

🤖 AI-Powered Descriptions
Generate compelling, SEO-optimized vehicle descriptions automatically using advanced AI technology. Simply select a vehicle, and the extension creates professional listings that attract buyers.

📸 Intelligent Photo Matching
Automatically match vehicle photos to inventory using VIN matching technology. The extension intelligently pairs photos from your photo library with the correct vehicle data.

📊 CSV Integration
Seamlessly sync with your inventory management system via CSV files. The extension automatically fetches the latest vehicle data, keeping your listings current without manual updates.

⚡ Auto-Sync & Updates
Set up automatic CSV synchronization that updates your inventory in the background. The extension checks for changes and updates listings automatically, ensuring accuracy.

🎯 Streamlined Workflow
Create professional Facebook Marketplace listings in minutes instead of hours. The extension handles the tedious work, so you can focus on selling.

🔒 Secure & Private
All data processing happens locally in your browser. CSV data and settings are stored securely on your device. AI API keys are encrypted and never shared.

PERFECT FOR:
- Automotive dealerships
- Used car lots
- Vehicle resellers
- Inventory managers
- Sales teams

HOW IT WORKS:

1. Upload or sync your vehicle inventory CSV file
2. Select vehicles from your inventory gallery
3. Match photos to vehicles automatically
4. Generate AI-powered descriptions with one click
5. Post directly to Facebook Marketplace

SETUP:

The extension requires:
- Vehicle inventory CSV file (with VIN, make, model, year, etc.)
- AI service API key (OpenAI or Anthropic) for description generation
- Access to your SFTP server or CSV file for inventory sync

SUPPORT:

For setup assistance, configuration help, or questions, please refer to the included documentation or contact support.

PRIVACY:

This extension processes data locally in your browser. Vehicle data and photos are not transmitted to third parties except for AI description generation (using your API key). All CSV data remains on your device or your designated server.

PERMISSIONS EXPLAINED:

- activeTab: To interact with Facebook Marketplace listing pages
- storage: To store your inventory CSV data and extension settings locally
- scripting: To inject necessary functionality into Facebook Marketplace pages
- sidePanel: To display the extension interface
- alarms: For automatic CSV refresh functionality
- host_permissions: To fetch CSV data from your server and communicate with AI services (using your API keys)

VERSION HISTORY:

v1.0.0 - Initial release
- Multi-store inventory support
- AI-powered description generation
- Automatic CSV synchronization
- Intelligent photo matching
- Facebook Marketplace integration
```
